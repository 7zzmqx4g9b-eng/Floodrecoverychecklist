
  # Flood Recovery Checklist

  This is a code bundle for Flood Recovery Checklist. The original project is available at https://www.figma.com/design/YhyVd7gD17hXf9x8NxWr0D/Flood-Recovery-Checklist.

  by Kobie Arayatakul 
  Flood Recovery Checklist

  A practical, step‑by‑step checklist to help individuals, families, or small teams recover after a flood.
  You can adapt this list for documentation, issues, or project boards in GitHub.

  

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
